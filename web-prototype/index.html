これは何のページ？

Dormire の姿勢計測機能を、スマートフォンのブラウザだけで動かせるか検証するためのプロトタイプです。開発チームの Go / No-Go 判定 に使う目的で公開している内部向けページであり、一般ユーザー向けのサービスではありません。

既存の iOS ネイティブ版と同じ 20 項目を、ブラウザ上の骨格認識エンジン (MediaPipe Pose Landmarker) で算出できるかを確認します。

使っている技術

  • HTML / CSS / JavaScript (ES modules) — フレームワークやビルドなし
  • Google MediaPipe Pose Landmarker (WebAssembly、端末内で実行)
  • Netlify (静的ホスティング、ビルド不要)

プライバシーと取り扱い

  • カメラ映像はこの端末のブラウザ内で処理されます。
  • 画像ファイル・計測値・個人情報はサーバーに送信・保存されません。
  • クッキーやトラッキングは使用していません。外部通信は MediaPipe のモデルファイル取得のみです。
  • ログインやアカウント登録、決済、連絡先収集は一切ありません。

計測の流れ

  1. 下のボタンから計測ページへ進む
  2. カメラ権限の許可を求められるので許可
  3. 全身がフレームに収まるように立つ（3〜4 m 離れる）
  4. 骨格が描画されたら「確定」で 20 項目の結果を表示
同意して計測ページへ進む
ボタンを押すまでカメラ権限は要求されません。
